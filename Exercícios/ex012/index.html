Inserindo Vídeos no Site

Incorporação de Vídeos em Hospedagem Própria

A inclusão de vídeos no site deve ser bem estudada pois, o carregamento de vídeo diretamente do seu site pode fazer com que o site fique muito pesado e como já estudamos, isso prejudica nossa página nos mecanismos de busca.

Vamos ver diversas maneiras de incluir vídeos no nosso site?!

Antes vamos falar sobre compatibilidade, no caso de navegadores os formatos mais compatíveis são:

  • .MP4
  • .M4V
  • .WEBM
  • .OGV

Caso o cliente ou você mesmo queira um vídeo hospedado no seu servidor do site, temos uma ferramenta para compressão e conversão de vídeo, caso este seja muito grande e incompatível para que rode em HTML, que é o HANDBRAKE, uma outra ferramenta gratuita assim como o GIMP.

Só uma obsevação, o HandBrake não converte vídeos em .OGV.

E caso o vídeo não seja autoral e você precise de algo pronto ou até mesmo imagens gratuítas, podemos usar os vídeos e imagens do PEXELS.

É importante ter o vídeo em vários formatos pois, pode ocorrer de o vídeo funcionar em um navegador e não funcionar em outros navegadores.

Bom agora vamos utiliza as tags para incluir um vídeo que está hospedado localmente, ou seja, no servidor de hospedagem do site.

E então deu certo? Muito bem, para isso tivemos que usar a tag video em sua maneira mais simples conforme está abaixo e esplicaremos alguns parâmetros que usamos para que o vídeo pudesse rodar normalmente.

A tag ficou da seguinte forma:

<video src="midia/golfinhos.mp4" width="560" height="315" controls></video>

Assim como com as imagens e também como com os áudios, foi necessário efetuar a inserção de alguns parâmetros para que tudo funcione corretamente, conforme mencionado abaixo:

  • <video src=""></video>, é a parte principal da tag, onde incluímos o local onde o vídeo está armazenado;
  • width="" é o parâmetro para definirmos a largura de exibição do vídeo;
  • height="" é o parâmetro para definirmos a altura de exibição do vídeo;
  • controls, é o parâmetro para que os controles do vídeo sejam exibidos e possamos colocar o vídeo para rodar.

Outra maneira de incluírmos vídeo também usando a tag video é a que aparecerá na exibição abaixo:

A tag para o vídeo acima fica dessa forma:

<video width="560" height="315" controls>
<source src="midia/golfinhos.mp4" type="video/mp4">
</video>

Dentro da tag, podemos disponibilizar todos os de arquivo de vídeo e de formatos compatíveis com todos os navegadores com o comando <source src="" type="">.

Outra coisa que podemos colocar é uma thumbnail, pois quando carregamos a página o vídeo estará parado no primeiro frame dele.

O vídeo com a thumb ficará assim:

Legal né!? O código para deixar o vídeo com a thumb ficou desse jeito:

<video width="560" height="315" poster="imagens/golfinhos_capa.png" controls>
<source src="midia/golfinhos.mp4" type="video/mp4">
</video>

Essas são as maneiras de inserir vídeos no site com os vídeos hospedados no servidor do mesmo.
Este não é o meio mais indicado até porque o fluxo de banda de trafego no site fica gigantesto e isso pode gerar um custo enorme para você ou para seu cliente.
Aprenderemos uma maneira mais barata de incluir vídeos no site para o custo ficar menor.

Incorporação de Vídeos Externos

Chegamos a forma mais "barata" de incluir um vídeo no nosso site, que é adicionando vídeo de uma fonte externa, ou seja, de um local de hospedagem próprio de vídeos, e uma das fontes dos mais conhecidas é o YouTube.

Abaixo tem um belo exemplo de vídeo vindo direto do YouTube:

O que fizemos para incluir o vídeo acima foi muito simples, e vamos colocar o passo a passo abaixo:

  • Entramos no YouTube;
  • Escolhemos o vídeo, e clicamos para executá-lo;
  • Clicamos em compartilhar e depois em incorporar;
  • Copiamos o código sem fazer alterações;
  • Colamos o código no VSC.

Simples né?! Melhor ainda se o cliente que você tem possuir um canal no YouTube, pois ao invés de hospedar o arquivo de vídeo no servido do site do cliente, o que pode gerar um custo absurdo como já falamos, podemos hospedá-lo na fonte externa e incorporá-lo ao nosso código.
