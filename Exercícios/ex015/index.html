Iniciando Estudo das CSS.

Vamos primeiro mostrar o jeito de como não se deve fazer a aplicação das CSS.

Existe um jeito (antigo) de se fazer a inclusão das que pode ser que funcionem, que pe colocando os estilos direto nos paragrafos, porém, os parâmetros não estarão mais de acordo com o que funciona atualmente, darei o exemplo abaixo.

<body bgcolor="red">
<h1> Olá, mundo!&/h1>
<p><font color="white">Oi!</font></p>

O uso do parâmetro, da propriedade, do valor, da tag, está erra pois é um método usando em HTML 4.

Explicação

Aplicando as CSS Externos.

Neste exercícios estamos utilizando as CSS Externas, o que fazemos é criar um arquivo em .css, nele nos colocamos apenas declarações de estilo que queremos, e também colocamos as regras necessárias.

Já na área head do nosso código criamos um link para o arquivo que neste caso chamamos de style.css, e ao inves de fazermos a alteração do estilo página por página, inserimos o link e alteramos todo o estilo das CSS no arquivo que foi criado, assim conseguimos fazer a alteração em todas as páginas que tem o link para o arquivo.

A área head ficou assim:

        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>CSS - Estilos Externos</title>
            <link rel="stylesheet" href="style.css">
        </head>

O arquivo style.css, ficou com o seguinte código: 

               @charset "UTF-8";
                body {
                    background-color: lightgoldenrodyellow;
                    font-family: 'Courier New', Courier, monospace;
                    font-size: 16px;
                    }
                h1 {
                    color: darkblue;
                    }
                h2 {
                    color:brown;
                    }
                p {
                    text-align: justify;
                    }

O arquivo style.css como falamos anteriormente leva apenas os seletores e dentro de cada seletor nós colocamos as declaração entre "{ }" (chaves), lembrando que declarações são o conjunto de propriedades e valores, a diferença aqui é que também configuramos regras, e elas sempre são configuradas no inicio do arquivo .css, a aplicada aqui foi "@charset "UTF-8";", mas veremos sobre regras em outro momento.

Inclusive podemos colocar mais de um link para chamar mais de um estilo para o site, além de podermos misturarmos as css inlines, internos e externos.

Devemos lembrar também que existe uma hierarquia nas css, sendo que sempre termos prioridade nesta mesma ordem, inline, interno ou local, externo por último.
