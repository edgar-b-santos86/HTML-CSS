Título 1

Desta vez faremos nosso modelo de site e colaramos um básico das CSS, mas que já vai mudar e muito nosso site, e isso com o que aprendemos até agora e mais algumas coisinhas para deixar ele mais interessante.

Colocaremos a tag <main> </main> no nosso html, apenas para dizer que o conteúdo dentro da área <body> </body>, é o conteúdo principal

Vamos usar as CSS no estilo externo, e veja abaixo como ficou nosso arquivo:

                    
                        @charset "UTF-8";
                        * {
                            font-family: Arial, Helvetica, sans-serif;
                        }
                        body {
                            background-image: linear-gradient(to top, #E3D4FF, #AC80FF);
                        }
                        main {
                            background-color: white;
                            border-radius: 10px;
                            box-shadow: 5px 5px 15px #24153f86;
                            width: 600px;
                            padding: 10px;
                            margin: auto;
                        }
                        h1 {
                            color: #24153F;
                            text-align: center;
                            text-shadow: 1px 1px 2px #150c25ab;
                        }
                        h2 {
                            color: #301269;
                        }
                        p {
                            text-align: justify;
                        }

Vamos relacionar item por item para sabermos o que cada um fez em nosso código:

  • @chaset "UTF-8" - Essa é nossa regra para que não tenhamos problemas com caracteres especiais;
  • * - Aqui colocamos nossas configurações globais;
  • background-image - Nesta declaração criamos o nosso gradiente no site;
  • background-color - Aqui colocamos a cor de fundo do nosso conteúdo;
  • border-radius - Esta arredonda as bordars onde colocamos nosso conteúdo;
  • box-shadow - aqui colocamos o sombreamento da caixa onde inserimos o nosso conteúdo;
  • width - determinamos o tamanho da área main;
  • padding - aqui definimos o acolchoamento do conteúdo, ou seja, o espaço ficará entre o conteúdo e a área delimitada para ele;
  • margin - aqui determinamos conteúdo seja centralizado mesmo que a página seja minimizada e maximizada, o alinhamento do conteúdo não será afetado;
  • text-align - para o alinhamento do texto;
  • text-shadow - para incluir sombreamento no texto.
  • *, body, main, h1, h2, p - esses foram os seletores utilizados.
