Trabalhando com Listas

Listas Ordenadas

  1. Acordar
  2. Escovar os dentes
  3. Passar a roupa
  4. Orar
  5. Ungir minha esposa e o bebê
  6. Pegar a marmita
  7. Ir trabalhar

Em listas ordenadas criamos primeiro a lista com a tag <ol> </ol>, e depois incluímos os ítens da lista com a tag <li>.

As próprias tags levam os nomes aqui citados <ol> Ordered List ou Lista Ordenada, e <li> Item List ou Item de Lista.

A partir do HTML5 o fechamento da tag <li> com </li> é opcional, entretanto, o VSC faz o fechamento da tag automáticamente.

Para personalizar a lista ordenada, podemos incluir um parâmetro na abertura da taag <ol>, o type.

O parâmentro type, tem três tipos de marcadores para as listas, os númericos, os iniciados em letras, e os que utilzam os algarísmos romanos.

A abertura da tag fica dessa forma <ol type="1">, para marcadores com números:

  1. Acordar
  2. Escovar os dentes
  3. Passar a roupa

<ol type="A">, para marcadores com letas maiúsculas:

  1. Acordar
  2. Escovar os dentes
  3. Passar a roupa

<ol type="a">, para marcadores com letas minúsculas:

  1. Acordar
  2. Escovar os dentes
  3. Passar a roupa

<ol type="I">, para marcadores com algarísmos romanos maiúsculos:

  1. Acordar
  2. Escovar os dentes
  3. Passar a roupa

<ol type="i">, para marcadores com algarísmos romanos minúsculas:

  1. Acordar
  2. Escovar os dentes
  3. Passar a roupa

Listas não ordenadas

Da mesma forma que criamos as listas ordenadas, também criamos as não ordenas, porém, usando as tags <ul> </ul> e depois incluímos os ítens nas tags <il>

Estas, entretanto, são listas que a ordem dos ítens não fara diferença para a sua criação, como uma lista de supermercado por exemplo.

Exemplo de Lista não ordenada:

Lista de Supermercado

  • Pão
  • Leite
  • Açucar
  • Arroz
  • Feijão
  • Carne
  • Tomate
  • Alface
  • Coentro
  • Alho
  • Cebola

Também podemos editar o tipo de marcador das listas não ordenadas, e o parâmetro type, e temos três tipos de símbolos, círculo, disco, e quadrado.

A abertura da tag fica dessa forma <ul type="circle">, para cículo:

  • Pão
  • Leite
  • Açucar

A abertura da tag fica dessa forma <ul type="disc">, para disco:

  • Pão
  • Leite
  • Açucar

A abertura da tag fica dessa forma <ul type="square">, para quadrado:

  • Pão
  • Leite
  • Açucar

Misturando listas

  1. Antigas
    1. Clipper
    2. Visula Basic
    3. Fortran
    4. Delphi
  2. Novas
    1. PHP
    2. Python
    3. JavaScript

É possível colocar uma lista ordenada dentro de outra lista ordenada, conforme foi feito acima, o código para que isso aconteça fica da seguinte forma:

            <ol>
                <li>Antigas</li>
                <ol type="a">
                    <li>Clipper</li>
                    <li>Visula Basic</li>
                    <li>Fortran</li>
                    <li>Delphilt;/li>
                </ol>
                <li>Novas</li>
                <ol type="a">
                    <li>PHP</li>
                    <li>Python</li>
                    <li>JavaScript</li>
                </ol>
            </ol>

Também é possível dar continuidade na contagem de uma lista para outra conforma esta lista abaixo:

  1. Antigas
    1. Clipper
    2. Visual Basic
    3. Fortran
    4. Delphi
  2. Novas
    1. PHP
    2. Python
    3. JavaScript

Para que possamos fazer a contagem de lista apartir de um número ou letra específica, usamos o parâmetro start e o inicio da tag fica assim <ol type="a" start="5">

Sempre usaremos números no parâmetro start até porque a contagem da lista é númerica, mesmo se estivermos usando letras para organiza-las, e o código fica da seguinte forma:

            <ol>
                <li>Antigas</li>
                <ol type="a">
                    <li>Clipper</li>
                    <li>Visula Basic</li>
                    <li>Fortran</li>
                    <li>Delphilt;/li>
                </ol>
                <li>Novas</li>
                <ol type="a" start="5">
                    <li>PHP</li>
                    <li>Python</li>
                    <li>JavaScript</li>
                </ol>
            </ol>

Lista de definições

As lista de definições são compostas por três tags, a principal para criar a lista, como o próprio nome diz "Definition List" a tag <dl> </dl>, o primeiro item é inserido pela tag <dt> </dt>, que é o termo, e a seguinte que é a definição usando a tag <dd> </dd>

Abaixo um exemplo de lista de definição

HTML
Linguagem de marcação para criação de sites
CSS
Linguagem de marcação para a criação de desing do site
JavaScript
Linguagem de programação para a criação de interatividade de um site

Para chegarmos ao resultado acima o código ficou da seguinte forma:

            <dl>
                <dt>HTML</dt>
                <dd>Linguagem de marcação para criação de sites</dd>
                <dt>CSS</dt>
                <dd>Linguagem de marcação para a criação de desing do site</dd>
                <dt>JavaScript</dt>
                <dd>Linguagem de programação para a criação de interatividade de um site</dd>
            </dl>

Tudo isso pode ser personalizado com CSS.
