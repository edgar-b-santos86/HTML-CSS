Aplicando imagens de fundo.

Nesta aula tivemos a informação de como aplicar imagens de fundo na página que estamos criando.

Primeiramente vamos acessar o GitHub do professor Gustavo Guanabara para baixar o pacote de imagens para trabalharmos com a aplicação delas em nosso plano de fundo do sites.

Criamos três <div></div> colocamos bordas e demos a elas a class="quadrado".

Após ter criado as <div></div> iniciamos a configuração das três da seguinte forma em nossas CSS internas:

                div.quadrado{
                   border: 1px solid black;
                    width: 300px;
                   height: 300px;
                   display: inline-block;
                   background-color: lightgray;
                   border-radius: 10px;
            }

Isso foi feito para que pudéssemos visualizar todos os três quadros em nossa página e assim efetuar a configuração deles.

Em seguida criamos a três id's diferentes para cada uma das div's, para que assim pudessemos aplicar imagens de fundo diferentes, e assim ficaram elas:

                <div class="quadrado" id="q1">
                </div>
                <div class="quadrado" id="q2">
                </div>
        
                <div class="quadrado" id="q3">
        
                </div>

Em seguida, aplicamos as seguintes configurações de CSS para cada uma delas:

                div#q1 {
                    background-color: skyblue;
                }

                div#q2 {
                    background-image: linear-gradient(to right, red, yellow);
                }

                div#q3 {
                    background-image: url('imagens/pattern003.png');
                }

Como podemos notar as configurações das divs 1 e 2 já são meio que conhecidas, pois já usamos o background-image em outras ocasiões e inclusive no nossa página do desasfio 10, entretanto, note que o consorciado a terceira div tem algo diferente, ao invés de usamos um valor como linear-gradient para que a cor variasse de uma para outra, usamos o URL e escolhemos uma das imagens baixadas para nossas aulas, e isso fez com que a imagem virasse o plano de fundo de nossa div.

Essa configuração simples faz com que a imagem seja carregada em seu tamanho total, e ainda veremos mais configuração para background neste capítulo.
