Título

Nesta aula iremos incluir arquivos de fontes dentro da pasta do nosso site, até porque as vezes o cliente já tem uma fonte pré-definida para utilizar no site ou em títulos específicos das páginas dele.

A primeira coisa que devemos fazer, não é instalar as fontes no nosso PC, nós vamos baixar a fonte e fazer um font type personalizado.

Primeiro vamos criar o nosso style local, e para baixar a fonte o professo Gustavo Guanabara indicou o site http://www.dafont.com/, e devemos ter cuidado ao escolher a fonte pois algumas delas não tem acentuação.

Após baixar a fonte e savar ela em uma pasta dos aquivos do site que estamos criando para que está fonte seja carregada pois ela também deve estar no servidor.

Temos as seguintes extensões de arquivos de fontes:

  • TrueType (.ttf)
  • OpenType (.otf)
  • Web Open Font Format (.woff)
  • Embedded Open Type (.eot)
  • Scalable Vector Graphics (.svg)
  • truetype-aat

para aplicar a fonte, devemos criar uma regra com um seletor especial conforme abaixo: 

            
            <style>
            @font-face {
                font-family: 'Cap';
                src: url('fonts/Capsmall.ttf') format('truetype');
                font-weight: normal;
                font-style: normal;
                }
            @font-face {
                font-family: 'CursiveSerif-Book';
                src: url('fonts/CursiveSerif-Book.ttf') format('truetype');
                font-weight: normal;
                font-style: normal;
                }
            body {
                font-family: Arial, Helvetica, sans-serif;
                }
            h1 {
                font-family: 'Cap', sans-serif;
                font-size: 3em;
                font-weight: normal;
              }
            p {
                font-family: 'CursiveSerif-Book', sans-serif;
                font-weight: normal;
               }
            </style>

Quando colocamos uma fonte externa nas CSS podemos dar um nome a ela como vimos acima, neste caso usamos duas regras de fontes diferentes para o nosso exercício.
