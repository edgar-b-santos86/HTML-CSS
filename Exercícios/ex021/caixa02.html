Como eram usadas as grouping tags.

As tags de agrupamento basicamente sempre foram duas a div e span, onde a div é uma block-level e a span é uma inline-level, e sempre eram usadas divs para a divisão das caixas dos sites, e colocávmos dentro delas as ids, para separação e posteriormente efetuar a estilizaçao com as CSS.

Porém, com a evolução do html e das css, agora temos as grouping tags semanticas como abaixo:

  • header
  • main
  • footer
  • section
  • article
  • aside
  • nav
e assim deixaremos o nosso código pronto para que receba as CSS, assim como fizemos com essa página.
