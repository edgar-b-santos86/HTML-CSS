Testando carga de imagens

Abaixo você verá uma imagem que está na mesma pasta.

Podemos carregar imagens que estão em subpastas

Podemos também carregar imagens externas

Explicação

Para incluir uma imagem no código HTML, podemos usa a tag <img>.

Dentro do código HTML a tag ficaria dessa maneira:

<img src="logo_html5_200.png" alt="Logo HTML5">

Temos nesta tag duas instruções importantes, sendo ela o local onde a imagem deve ser encontrada e o o texto alternativo da imagem.

Sendo img src="" a imagem local onde deve ser buscada a imagem, seja interno ou exteno, e alt="" o texto alternativo da imagem.

Dentro do Visual Studio Code quando cursor estiver entre as aspas "" da parte do código img src="" apertamos Ctrl + espaço e se a imagem estiver na mesma pasta da página que está sendo desenvolvida, basta selecionar o nome da imagem e ela sera carregada.

Também podemos incluir uma imagem de um site externo img src="" colando o link da imagem entre as aspas.
