Na primeira aula do curso em vídeo do professor Gustavo Guanabara, aprendemos a criar o código padrão para inicio de toda página HTML5 como segue abaixo.

    <!DOCTYPE html>
    <html lang="pt-Br">
    <head>
        <meta charset="UTF-8">
        <meta http-equiv="X-UA-Compatible" content="IE=edge">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Meu Primeiro Exercício</title>
    </head>
    <body>
    
    </body>
    </html>

O código acima é criado apenas digitando "!" e apertando o enter, isso usando o Visual Studio Code.
