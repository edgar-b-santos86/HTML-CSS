Iniciando Estudo das CSS.

Vamos primeiro mostrar o jeito de como não se deve fazer a aplicação das CSS.

Existe um jeito (antigo) de se fazer a inclusão das que pode ser que funcionem, que pe colocando os estilos direto nos paragrafos, porém, os parâmetros não estarão mais de acordo com o que funciona atualmente, darei o exemplo abaixo.

<body bgcolor="red">
<h1> Olá, mundo!&/h1>
<p><font color="white">Oi!</font></p>

O uso do parâmetro, da propriedade, do valor, da tag, está erra pois é um método usando em HTML 4.

Aplicando as CSS Locais/Internas.

Neste exercícios estamos utilizando as CSS Locais/Internas, o que fazemos é criar uma tag <style> </style> dentro da área <head> </head> do nosso código, e como podemos percerber o estilo já está aplicado a essa página.

A aplicação do código ficou dessa forma:

        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>CSS - Estilos Locais/Internos</title>
            <style>
            body {
                background-color: lightgray;
                font-family: Arial, Helvetica, sans-serif;
                font-size: 16px;
            }
            h1 {
                color: darkblue;
                background-color: lightblue;
            }
            h2 {
                color: darkred;
            }
            p{
                text-align: justify;
            }
            </style>
        </head>

Como podemos verificar colocamos a tag style abaixo da tag title, onde colocamos seletores dentro desta tag para cada local que queremos colocar o estilo escolhido. No exemplo acima que aplicamos nesta página os seletores foram as tags body, h1, h2, p, e dentro de cada seletor nos colocamos as declaração entre { }, lembrando que declaração são o conjunto de propriedades e valores.

Essa técnica de aplicação das CSS de duas desvantagens, apesar de ser muito melhor que a técnica inline, uma delas é que o estilo vai ocupar muito mais linhas em alguns casos, do que o próprio código, e nesse caso, quando você abrir o VSC, você vai ter muitas linhas antes de chegar de fato ao código.

A outra desvantagem é a seguinte, muito mais incomoda, é que se o site tiver muitas outras páginas você terá que colar o estilo em todas as páginas e, lembrando novamente do exemplo que demos antes se o cliente em alguma época de festas quiser fazer alguma alteração na cor das letras ou no fundo, neste caso teremos que fazer a alteração no estilo e novamente ter que ir colocando em todas as outras.
