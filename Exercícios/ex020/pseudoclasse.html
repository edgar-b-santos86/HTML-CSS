Pseudo-Classes e Pseudo-Elementos

Pseudo-Classes

As pseudo-classes são palavras-chaves adicionadas as declarações de um seletor em CSS usadas para definir o estado especial de um elemento. Elas são precedidas por dois pontos (:).

  • :hover - Aplica estilos quando o mouse passa sobre o elemento.
  • :active - Aplica estilos quando o elemento está sendo clicado.
  • :focus - Aplica estilos quando o elemento recebe foco, como um campo de entrada.
  • :visited - Aplica estilos a links que já foram visitados.
  • :first-child - Seleciona o primeiro filho de um elemento pai.
  • :last-child - Seleciona o último filho de um elemento pai.

Pseudo-Elementos

Já os pseudo-elementos em CSS são palavras-chaves adicionadas a declaração de um seletor e permitem que você formate um pedaço específico de um elemento referenciado. Eles são precedidos por dois sinais de dois pontos (::).

  • ::before - Insere conteúdo antes do conteúdo de um elemento.
  • ::after - Insere conteúdo após o conteúdo de um elemento.
  • ::first-line - Aplica estilos à primeira linha de um bloco de texto.
  • ::first-letter - Aplica estilos à primeira letra de um bloco de texto.

Criamos as classes exemplo e especial, a primeira aplicamos para os dois primeiros links, e a classe especial aplicamos apenas para o segundo link você notou a diferença?

Neste caso as nossas declarações para a classe especial, foram para fazer com que a fonte ficasse em negreto (bold), a cor para links não visitados ficassem em vermelho, e para que ao passarmos o mouse sobre o link, o sublinhado aparecesse, além disso, nos links visitados a cor mudaria para vermelho escuro.

            
                .exemplo {

                    text-decoration: none;

                    color: #700000;
                }
                .exemplo:hover {
                    text-decoration: underline;
                }
Essas foram as pseudo-classes que utilizamos para nossos dois primeiros exemplos de links.

Já para a classe especial utilizamos uma pseudo-classe para que ao passar o mouse em cima do link a letra ficaria branca, o fundo ficaria preto, e o sublinhado sairia, também colocamos pseudo-elementos antes e depois do link.

            
                .especial:hover {
                    color: #ffffff;
                    background-color: #000000;
                    text-decoration: none;
                }
                .especial::before {
                    content: '>>';
                    font-weight: lighter;
                }
                .especial::after {
                    content:'<<';
                    font-weight: lighter;
                }

Conteúdo Especial

Vamos criar mais um exemplo, e neste nós usaremos um bloco especial com uma <div>, e uma vantagem das divs é que podemos incluir outras tags dentro delas.

Passe o mouse aqui.

SURPRESAAAAAA!!!

Veja o conteúdo oculto aparecendo!

A tag acima ficou assim: 

            
                <div>
                    Passe o mouse aqui.
                    <p>SURPRESAAAAAA!!!</p>
                </div>

E para que ela funcione corretamente nós incluímos um estilo CSS para a div e seus elementos internos.

            
                div > p {
                    display: none;
                }
                div:hover > p {
                    display: block;
                    background-color: #ffff00;
                }

E lembrando, os sinais que vimos até agora nas últimas aulas:

  • # = id
  • . = class
  • : = pseudo-classe
  • :: = pseudo-elemento
  • < = seletor de filho
  • > = seletor de descendente
  • * = universal
Estes são nossos seletores personalizados.
