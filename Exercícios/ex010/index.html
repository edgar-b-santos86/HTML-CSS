Trabalhando com Links Externos

Para Criar um link externo devemos a tag <a href=""> </a>, a tag <a> vem de âncora e dentro do do parâmento href="" da tag, devemos inserir o link completo do site que queremos acessar conforme os exemplos abaixos:

Acesse o meu canal no Youtube clicando aqui

Acesse também o site do Curso em Vídeo e o Canal do YouTube do Curso em Vídeo

Por sinal esse conteudo está sendo criado enquanto faço os cursos do Curso em Vídeo se inscreva lá!!!

No exemplo acima o resultado será esse:

        Acesse o meu canal no Youtube clicando <a href="https://www.youtube.com/@edgarbarbosadossantos1237">aqui</a>;
        Acesse também o site do <a href="https://www.cursoemvideo.com/login/">Curso em Vídeo</a> o Canal do 
        <a href="https://www.youtube.com/@CursoemVideo">YouTube do Curso em Vídeo</a>

Fazendo o código acima, quando o visitante clicar o texto com HyperLink, ele será direcionado para o site indicado, entretanto, isso fará com que o link clicado carregue sobre sua página e talvez esse usuário não volte a entrar no seu site.

Para sanar este problema usamos os seguintes parâmetros na tag <a>, o target que significa alvo e o rel de relacionado.

Dentro do parâmetro target podemos inserir o target="_blank", target="_self", target="_top" e target="_parent" sendo eles para:

target="_blank":
Irá abrir a página do link em uma janela ou aba em branco.
target="_self":
Irá abrir a página na janela ou aba atual.
target="_top":
vai desfazer todos os frames e abrir o link destino no navegador completo.

Dentro do parâmetro rel podemos colocar rel="external", rel="next" e rel="prev", sendo cada um servido para o seguinte propósito:

rel="external":
Indica que o é de um site externo que não fa parte do site que está.
rel="next":
Indica que o link é para a próxima parte do documento atual.
rel="prev":
Indica que o link é para a parte anterior do documenti atual.

Temos bastante conteúdo acima mas, vamos ao exemplo de como fazer para que a página não seja carregada sobre a sua.

Tente acessar agora os links abaixo, parecem ser o mesmo mas o comportamento de abrir a página será diferente:

Acesse o meu canal no Youtube clicando aqui

Acesse também o site do Curso em Vídeo e o Canal do YouTube do Curso em Vídeo

Agora que foi solucionado o problema e o link externo abri em outra aba ou janela, veja abaixo como ficou o código:

        Acesse o meu canal no Youtube clicando <a href="https://www.youtube.com/@edgarbarbosadossantos1237" target="_blank" rel="external">aqui.</a>;
        Acesse também o site do <a href="https://www.cursoemvideo.com/" target="_blank" rel="external">Curso em Vídeo</a>
        e o Canal do <a href="https://www.youtube.com/@CursoemVideo" target="_blank" rel="external">YouTube do Curso em Vídeo</a>.

O parâmetro rel é importante pois ele indica para os buscadores de internet como o google, algumas informações que não serão apresentadas para o usuário.

Não falomos sobre os as propriedades do rel="nofollow", rel="dofollow" e rel="author", então segue abaixo:

rel="nofollow":
Indica que é um link para um site externo, patrocinado, entrentato, que eu não quero que tenha relevância nos sites de busca, ou seja, quando alguém fizer uma busca sobre o site que estamos indicando, ele não será relevante nos resultados do buscador, é um não siga em tradução direta.
rel="dofollow":
Indica que é um link para um site externo, patrocinado, e que diferente do "nofollow" ele indica o link nos sites de busca, em tradução direta seria um siga.
rel="author":
Indica que é um link para o site do autor do artigo atual.

Links Internos

Aprender a criar links é importante pois, ao criar um site você precisará muito de links para que o usuário possa navegar entre as páginas dele.

Veremos dois meios de criar links internos, abaixo o primeiro exemplo:

Clique no aqui para acessar a segunda página do nosso site.

clicou no link, entrou na nossa segunda página e voltou?
Se você fez isso então segue abaixo o modo de como ficou o código:

        Clique no <a href="pag002.html">aqui</a> para acessar a segunda página do nosso site.

No metodo de criação de link interno que vimos acima, não foi preciso o URL ou endereço dentro da tag <a href="">, isso porque o arquivo da segunda página está localizado dentro da mesma pasta que o arquivo da página index, que a nossa página principal.

Dica importante, no Visual Studio Code ou VSCode, quando você está com o cursos dentro da tag <a href="">, basta apertar CTRL + ESPAÇO, ele já te mostra quais os arquivos de página estão na mesma pasta da página que você está trabalhando.

Mais acima falamos sobre os parâmetros target e rel, inclusive sobre o rel="external", quando fazemos referência de que um link é de uma página externa, entretanto, não existe o rel="internal", colocaremos abaixo novamente o link para a segunda página, porém, mais completo.

Então Clique Aqui para acessar nossa segunda página novamente.

E ai clicou? Bom se clicou viu que o resultado foi o mesmo né!?

A diferença está no código que ficou conforme demonstrado abaixo:

        <a href="pag002.html" rel="next">Clique Aqui</a> para acessar nossa segunda página novamente.

Notou a diferença no código? Usamos o rel="next", para informar que o link está relacionado a próxima página de navegação, e no link voltar da página 02 usamos o rel="prev" de previous ou anterior.

Acredito que nessa altura você possa estar se perguntando, e o que aconteceu com o parâmentro target? Pois é, não colocamos, porém, não esquecemos dele.

Quando falamos de links internos, se não colocarmos o parâmetro target o link que clicarmos vai abrir na mesma aba ou janela, porém, o código não fica completo, neste caso, devemos usar o parâmetro target="_self" no nosso link.

Clique aqui, para acessar nossa segunda página novamente.

Vistou nossa segunda página novamente? Notou alguma diferença? Acredito que não, porém, a diferença está no código, que agora sim está completo, você consegue verificar como ele ficou conforme abaixo:

        Clique <a href="pag002.html" target="_self" rel="next">aqui</a>, 
        para acessar nossa segunda página novamente.

Neste caso usamos o target="_self" para que o link fosse aberto na mesma página, diferente do que fazemos com links externos, onde queremos que a página abra em uma janela ou aba diferente e usamos target="_blank".

O segundo modo de criar um link interno é parecido com a criação de links externos, sendo necessário inserir todo o caminho ou a URL de onde está a página.

Bom, clique aqui para acessar a terceira página.

Para esse link, mesmo sendo de uma página interna do nosso site, foi necessário digitar o caminho ou URL, conforme falamos acima, pois o arquivo está em uma pasta diferente, veja como o link ficou abaixo:

        Bom, <a href="EX010_2/pag003.html" target="_self" rel="next">clique aqui</a>
        para acessar a terceira página.

A dica que demos anteriormente de, apertar CTRL + ESPAÇO, e o VSCode mostra o endereço ou o arquivo html que queremos vincular no link continua valendo, porém, para voltar da página três para a principal, o metodo é diferente.

Dentro da tag <a href="">, devemos digitar "..", isso mesmo dois pontos seguidos, assim o VSCode vai entender que o arquivo que queremos vincular o link está em uma pasta num nível anterior.

O código na página três para voltar para a principal ficou assim:

        <a href="../index.html" target="_self" rel="prev">Clique aqui</a> para voltar a página principal.

Se o arquivo ou página que queremos vincular no link está mais de um nível ou pasta acima digitamos "...", e almentamos o número de pontos de acordo com o nível ou número de pastas acima que o arquivo que queremos está.

Links para Download.

Nesta no Curso em Vídeo, iremos fazer aprender a fazer links para downloads de arquivos, para isso na mesma pasta do exercício 10, uma outra pasta e colocaremos alguns arquivos, eu coloquei alguns PDFs de livros de programação.

Cada livro foi baixado e compactado, e mantivemos os PDFs, abaixo criaremos a lista de arquivos para Download.

  • Algoritimos e Programação em PDF.
  • Algoritimos e Programação em ZIP.
  • Apostila de HTML, CSS e JavaScript em PDF
  • Apostila de HTML, CSS e JavaScript em ZIP
  • Apostila de Java e Orientação a Objetos em PDF.
  • Apostila de Java e Orientação a Objetos em ZIP.
  • Lógica de Programação para Iniciantes em PDF.
  • Lógica de Programação para Iniciantes em ZIP.
  • Java Básico e Orientação a Objetos em PDF.
  • Java Básico e Orientação a Objetos em ZIP.

A criação de links de download funciona basicamente igual ao link interno a diferença é que selecionamos o arquivo que desejamos baixar, e além disso usamos os parâmetros download="" e entre as aspas colocamos o nome do arquivo, e type="" onde informamos o tipo de mídea.

Podemos verificar médea types no site http://iana.org, esse site tem todos os media types existentes.

Podemos verificar como ficaram os links para download conforme abaixo:

        <li><a href="Livros/algoritmos-e-programacao-ii.pdf" download="algoritmos-e-programacao-ii.pdf" type="application/pdf">Algoritimos e Programação em PDF.</a></li>
        <li><a href="Livros/algoritmos-e-programacao-ii.zip" download="algoritmos-e-programacao-ii.pdf" type="application/zip">Algoritimos e Programação em ZIP.</a></li>

Outra dica que deixo aqui e foi dada pelo Professor Gustavo Guanabara é o envelopar os textos para incluir as tags e para isso, selecionamos o texto e apertamos CTRl+Shift+P, selecionar abreviatura abb e inserir a teg que queremos colocar para aquele texto.
