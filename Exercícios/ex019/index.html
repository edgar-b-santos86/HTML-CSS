Seletores

Trabalhando com seletores

Como já vimos, fazemos as formatações (de forma semântica) para o nosso site por meio de seletores que indicam nas CSS, as tag que queremos "configurar" para os elementos do HTML, porém, as vezes queremos fazer diferentes formatações para dois elementos que fazem parte do mesmo grupo de seletores. Vamos ver mais a frente como fazemos isso.

Id (identificadores)

Seletores Personalizados.

Neste exercício iremos incluir identificadores e faremos nossas CSS no estilo externo para que possamos ter uma melhor visualização.

Como falamos um pouco acima, faremos seletores personalizados pois normalmente as nossa CSS são assim: 

                @charset "UTF-8";
                body {
                    background-color: rgb(162, 238, 162);
                    font-family: Arial, sans-serif;
                    }
                    h1 {
                    color: rgb(133, 45, 45);
                    }
                    h2 {
                    color: rgb(5, 82, 5)
                    }

Vendo as declarações de CSS assima sabemos que o corpo do site terá a cor verde, a fonte será arial (ou outra não serifada), os títulos de hierarquia 1 (h1) terão a cor marrom e os títulos de hierarquia 2 (h2) terão a cor verde escuro.

O problema ai é que se eu quiser que o primeiro h1 tenha uma cor ou letra diferente eu não conseguiria fazer isso não é mesmo? Acontece que podemos fazer isso sim incluindo um identificador (id) no nosso HTML e referenciando ele na nossa CSS.

Neste caso colocamos um identificador no nosso primeiro h1 (no HTML) e ele ficou assim: 

                
                    <h1 id="principal">Id (identificadores) e Classes</h1>
E a aplicaçao nas CSS ficou assim: 
                
                    h1#principal {
                        text-align: center;
                        color: black;
                    }
Assim nós conseguimos personalizar configurações ou formatações para determinados elementos do HTML, por meio das CSS, e o identificador no css não pode ter espaço.

Também podemos fazer o uso do identidicador nas CSS sem a necessidade de indicar o HTML, e o código ficaria igual ao que consta abaixo: 

                
                    #principal {
                        text-align: center;
                        color: black;
                    }

Isso acontece porque todo identificador (id) em HTML é igual a "#" na CSS, e toda class em HTML é igual a "." na CSS. E já já falaremos sobre class.

Então lembre-se: 

                id (HTML) = # (CSS)
                class (HTML) = . (CSS)

Class (classes)

Seletores Personalizados.

Ok! Mas agora eu quero formatar meu segundo h1 com a mesma configuração do primeiro, posso usar o mesmo id para os dois? NÃO, se você fizer isso estará fazendo errado. O W3C, diz que não é permitido ter ids duplicados em um mesmo documento HTML, porém, temos uma solução pra isso: as classes ou class.

Assim como o id a class é aplicada no HTML da seguinte forma abaixo: 

                
                    <h1 class="secundarios">Class (classes)</h1>
                
            
                
                    .secundarios {
                                background-color: rgba(226, 191, 103, 0.644);
                                }

Podemos utilizar mais de uma classe no mesmo elemento apenas separando elas por espaço, e na tag ela ficaria assim: 

                
                    <h1 class="secundarios terciarios">Class (classes)</h1>
E assim conseguimos aplicar mais de uma formatação no mesmo elemento de HTML, e inclusive podemos aplicar classes nos textos específico do conteúdo do nosso HTML utilizando uma classe dentro da tag <span> </span>.
