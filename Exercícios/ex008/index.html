Principais Formatações

Negrito ou destaque

Nesta frase temos um termo em negrito usando a tag <b> que é uma tag não semântica

Nesta frase temos um termo em destaque usando a tag <strong> que é uma tag semântica

Itálico ou Ênfase

Nesta frase termo um em itálico usando a tag <i> que não é semântica

Nesta frase temos um termo com ênfase usando a tag <em> que é semântica

Texto Marcado

Nesta frase temos uma marcada com a tag <mark> que é semântica

E temos outro texto marcado nesta frase

Texto Grande e Pequeno

Estamos criando um texto grande e um texto pequeno usando as tags <big> que já não é mais semântica, e a tag <small> que continua sendo semântica

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele pode ser lido mas não deve ser considerado, e para isso usamos a tag <del> </del>

Texto Inserido ou Sublinhado

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, que pode ser considerado sublinhado semântico, para isso usamos a tag <ins> </ins>

E existe também o texto sublinhado com a tag <u> que não é semântica

Texto sobrescrito

Para inserir algo do tipo x2+3 ou 520 e neste caso usamos a tag <sup>

Textto Subscrito

Para inserir algo do tipo H2O e neste caso usamos a tag <sub>
